# Eger Térkép App

Ez a repository az Eszterházy Károly Katolikus Egyetemen készített BSc szakdolgozatom forráskódját tartalmazza. Az alkalmazás egy modern, keresztplatformos okosturisztikai mobilalkalmazás, amely Eger városának felfedezését könnyíti meg.

## Főbb funkciók
* **Térképes keresés és szűrés:** Helyi látnivalók dinamikus lekérdezése kategóriák alapján.
* **Útiterv-építő:** Személyre szabott bejárási sorrend összeállítása és többpontos navigáció indítása.
* **Közösségi funkciók:** Látnivalók értékelése, valamint az elkészült útitervek publikus túraként történő megosztása.
